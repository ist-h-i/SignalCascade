# Implementation Task Archive

このディレクトリは、完了済みの実装タスク文書を退避するための場所です。
active docs と違って、ここにある文書は現在の Source of Truth ではありません。

## 置くもの

- 完了済みの migration plan
- 実行スケジュール
- 実装ログ
- reviewer handoff
- historical requirements

## 置かないもの

- 現在の運用手順
- 現在の canonical spec
- 現在の実装説明

## 運用ルール

- active docs から外した時点で、ここへ移動します。
- archive 内の文書は、原則として当時の内容を保存します。
- 再利用が必要になったら archive を直接更新せず、active docs に新しい責務の文書を作ります。
